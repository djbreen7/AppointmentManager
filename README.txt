# Appointment Manager

## Purpose of Application
Appointment Manager allows users to easily schedule appointments with customers and contacts.

## Reporting
In addition to reporting contact schedules and appointments by type and month, users can view a report of audit history
for customers that includes information about when the customer was created, last updated, and who performed each action.

## Running AppointmentManager

### Pre-Requisites
- Ensure you've added and configured the JavaFX SDK to IntelliJ: https://openjfx.io/openjfx-docs/#IDE-Intellij
- Ensure you've added and configured the Java JDK in IntelliJ: https://www.jetbrains.com/help/idea/sdk.html#jdk
- Ensure the MySQL Connector library is installed and configured
- Attempt to build the project and address any errors related to the dependencies mentioned above

### Running the program
1. Open IntelliJ
2. Click File > Open
3. Select the project folder
4. Once the project has loaded, click "Play" or press CTRL+F5
